Program to check which number is greater

# Algorithim
Enter Two Number a and b
Check if a is greater than b 
	if yes print a is greater
	else b is greater


Pseudo Code: 

    input int a and b
    check if a > b
	    print a is bigger
	else if b > a
		print b is bigger
	else if a equal to b
		print number are equal
	else 
		print critical error
